Typography is based on Material Design
It is using it's font size and color schemes to provide beautiful typography

Headers

To preserve semantic structures, you should style the <h1> - <h6> heading tags with the styling classes
Default <h1> - <h6> have some fallback styling, but it's encuraged to use .display-#

    SELECTOR / SASS
    PREVIEW
    HTML
  • .display-4 @include display-4

    Display 4

    
    <h1 class="display-4">Display 4</h1>
  • .display-3 @include display-3

    Display 3

    
    <h1 class="display-3">Display 3</h1>
  • .display-2 @include display-2

    Display 2

    
    <h1 class="display-2">Display 2</h1>
  • .display-1 @include display-1

    Display 1

    
    <h1 class="display-1">Display 1</h1>
